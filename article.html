Switched to github since it's free

by RaVeN
2/15/2024 | ~1 min read

I've had various webhost throughout the years and it's always some ridiculous amount of money for hosting. Everything I could've made in was basic html and never needed any CMS software or MySQL databases to support it.

So here I am on github!

It was either github or google drive; github in my head seemed like a more logical choice. Plus, google lately seems hellbent on deleting old, unused content. It would be my fear if I stepped away from working on the site and it would be deleted without my consent.
